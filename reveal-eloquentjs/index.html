Werte, Typen und Operatoren

Einleitung

In JavaScript und Java gibt es Werte und Typen. Beide Sprachen unterscheiden zwischen primitiven und komplexen Datentypen. In diesem Kapitel betrachten wir die Gemeinsamkeiten und Unterschiede.

Primitive Typen

Primitive Typen sind grundlegende, unveränderliche Werte in JavaScript:

  • Number
  • String
  • Boolean
  • null
  • undefined
  • Symbol

Unterschiede zu Java

Java hat primitive Datentypen wie int, float und boolean, die statisch typisiert sind.


							int number = 5;
							String text = "Hello";
							boolean isActive = true;
						

							let number = 5;
							let text = "Hello";
							let isActive = true;

Unterschied: JavaScript ist dynamisch typisiert und erlaubt flexible Typkonvertierungen.

Automatic Type Conversion

JavaScript versucht oft, Werte automatisch zu konvertieren, wenn unterschiedliche Typen aufeinandertreffen:


			console.log(5 + "5"); // "55"
			console.log(5 - "3"); // 2
			console.log(true + 1); // 2

Vergleichsoperatoren

Die automatische Typkonvertierung kann zu unerwartetem Verhalten führen, besonders bei ==:


			console.log(5 == "5"); // true (automatische Konvertierung)
			console.log(5 === "5"); // false (strikter Vergleich)

`typeof`-Operator

In JavaScript kannst du den Typ eines Wertes mit typeof überprüfen:


			console.log(typeof 42); // "number"
			console.log(typeof "Hello"); // "string"
			console.log(typeof null); // "object" (historischer Fehler)
			console.log(typeof undefined); // "undefined"

Funktionen

Funktionen sind grundlegende Bausteine in JavaScript. Sie ermöglichen Wiederverwendung von Code.

Datenstrukturen

Objekte und Arrays sind die zentralen Datenstrukturen in JavaScript, die für die Speicherung und Manipulation von Daten verwendet werden.

Höhere Funktionen

Höhere Funktionen sind Funktionen, die andere Funktionen als Argumente nehmen oder zurückgeben. Sie sind in JavaScript äußerst nützlich.
