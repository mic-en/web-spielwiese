Werte und Typen

JavaScript unterscheidet primitive Typen und Referenztypen:

  • Primitive Typen: Number, String, Boolean, null, undefined, Symbol
  • Referenztypen: Object, Array, Function

Operatoren


					console.log(5 + 3); // Addition
					console.log(5 == "5"); // Vergleich ohne Typprüfung (true)
					console.log(5 === "5"); // Vergleich mit Typprüfung (false)
